Frequently asked questions

Do I need an account?

No. Canopy has no accounts, no sign-up, and no login. Your sessions, streaks, coins, and stats are stored locally in your browser using Chrome's built-in storage. There is no server involved.

Can I use it without blocking any sites?

Yes. Site blocking is entirely optional. If your blocklist is empty, the timer runs on its own with ambient sounds and tree growth. Blocking only activates when you add sites to the list and start a focus session.

Does the timer keep running if I close the popup?

Yes. The timer runs in Chrome's background service worker and uses chrome.alarms for reliability. It survives popup closures, tab closures, and even Chrome restarts. Your session will be waiting for you when you come back.

Is this page saying Forest is bad?

No. Forest works well for a lot of people, especially those who use the mobile app and want the social and environmental features. This page exists for people who specifically want a Chrome-only focus timer without an account requirement. Different tools for different workflows.

Is Canopy free?

Yes. All features are free. There are no premium tiers, no feature gates, and no "upgrade" prompts.

What permissions does it need?

Storage, alarms, notifications, and audio playback. Site blocking requires an additional host permission that Chrome asks for the first time you start a session with blocking enabled. If you decline, the timer still works -- just without blocking.
